# INE Municipios

This project holds Spain cities and states retrieved from INE DB so you can use 
it in your project. It also include a Mysql database populated with data.

It is a starting point if you need to implement the export from the Ministry of the Interior for accommodations and vehicle rentals.

## Spanish translation

Este proyecto tiene un listado de municipios y provincias de España con la codificación
del INE de forma que puedas utilizarlos en tus proyectos. Incluyen una base de datos
Mysql con los datos.

Es un punto de partida si necesitas implementar la exportación del Ministerio del Interior
para alojamientos y alquiler de vehiculos.
